## koa 应用生成器(脚手架工具)  全局安装 只需要安装一次

1. npm install koa-generator -g


## 创建项目

1. koa2 demo

## 安装依赖

1. cd demo
2. npm install

## 启动项目

1. npm start

## 创建项目

1. express demo


## 安装依赖

1. cd demo
2. npm install


## 启动项目

1. npm start
